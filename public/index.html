Features

Everything you need to audit SSL certificates

A complete set of certificate inspection capabilities in one lightweight tool.

Security Grading

Grades certificates from A+ to F based on TLS protocol version and days until expiry. Instantly know if a site meets modern security standards.

Expiry Countdown

Visual ring chart showing remaining days before the certificate expires. Catch expiring certificates before they take your site down.

Issuer Verification

Displays the Certificate Authority (CA) that issued the certificate, their organization, and the full chain of trust from leaf to root.

Cipher Analysis

Shows the active TLS protocol (1.2 or 1.3) and the cipher suite negotiated during the handshake. Verify your site uses strong encryption.

SAN Coverage

Lists every Subject Alternative Name on the certificate. See exactly which domains and subdomains are covered by a single cert.

Expired Cert Detection

Detects expired and self-signed certificates that browsers would block. Test problem domains like expired.badssl.com to see it in action.

How It Works

From domain input to full report in 3 steps

01

Enter Domain

Type any domain name — with or without https:// prefix. The tool extracts the hostname automatically.

02

TLS Handshake

The server opens a TLS socket on port 443, performs the handshake, and pulls the raw certificate object from the connection.

03

Report Generated

Certificate data is parsed into a structured report: subject, issuer, SAN list, validity, cipher, chain, and security grade.
